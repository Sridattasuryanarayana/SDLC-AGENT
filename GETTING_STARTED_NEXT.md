## 🔧 Configuration for Real LLM (Not Mock)

Currently, system uses **Mock provider** (no API calls needed). For actual code generation:

### **Step 1: Create .env file**
```bash
# Copy template
cp .env.example .env

# Edit .env with your API keys
```

### **Step 2: Add OpenAI API Key**
```
# .env
LLM_PROVIDER=openai
OPENAI_API_KEY=sk-your-api-key-here
OPENAI_MODEL=gpt-4
```

Or for Anthropic:
```
# .env
LLM_PROVIDER=anthropic
ANTHROPIC_API_KEY=sk-your-api-key-here
ANTHROPIC_MODEL=claude-opus
```

Or for local Ollama:
```
# .env
LLM_PROVIDER=ollama
OLLAMA_URL=http://localhost:11434
OLLAMA_MODEL=llama2
```

### **Step 3: Restart system**
```powershell
# Stop current instance (Ctrl+C)
# Restart
python web_app.py
# or
python run_sdlc.py --watch
```

✅ **System will now use real LLM instead of mock!**

## 🌐 Configure GitHub PR Automation

To enable automatic PR creation and merging:

### **Step 1: Generate GitHub Token**
1. Go to https://github.com/settings/tokens
2. Click "Generate new token"
3. Select scopes: `repo:full`, `workflow`
4. Copy token

### **Step 2: Add to .env**
```
# .env
GITHUB_TOKEN=ghp_your_token_here
GITHUB_REPO=YourUsername/YourRepoName
GITHUB_BRANCH=main
```

### **Step 3: Configure Approvals (Optional)**
```
# .env
AUTO_MERGE_ENABLED=true
REQUIRED_APPROVALS=1
```

### **Step 4: Test**
```powershell

# Watch console for:
# - Branch created
# - Code committed
# - PR created
# - Auto-merge (if approved)
```

## ⚠️ Common Issues & Solutions

### **Issue: "Agent not found" error**
```
❌ Error: ModuleNotFoundError: No module named 'agents'
```
**Solution**: Make sure you're running from the SDLC-AGENT directory
```powershell
cd "c:\Users\srakella\OneDrive - The Estée Lauder Companies Inc\Desktop\SDLC Agent\SDLC-AGENT"
python web_app.py
```

---

### **Issue: JSON serialization error**
```
❌ Error: Object of type X is not JSON serializable
```
**Solution**: This is fixed in the latest version. Ensure you have latest code:
```powershell
git pull origin main
```

---

### **Issue: File watcher not triggering**
```
❌ Task status changed but nothing happened
```
**Solution**: 
1. Verify Excel file path is correct
2. Make sure column D has status="pending"
3. Save Excel file after each change
4. Check that watch mode is running: `python run_sdlc.py --watch`

---

### **Issue: GitHub PR not creating**
```
❌ Code generated but no PR appeared on GitHub
```
**Solution**:
1. Check GITHUB_TOKEN is valid: `echo $env:GITHUB_TOKEN`
2. Verify GITHUB_REPO is correct in .env
3. Check token has `repo` scope permissions
4. Look for errors in console output

### **Common Questions**

**Q: Why does it say "mock provider" in my output?**  
A: You haven't configured a real LLM yet. Edit `.env` with your API key.

**Q: Can I use this with my existing codebase?**  
A: Yes! Place your code in `workspace/` directory and agents will modify it.

**Q: How do I add my own agents?**  
A: Create new file in `agents/` following `base_agent.py` pattern, then add to orchestrator.

**Q: Can I use this with local LLM (Ollama)?**  
A: Yes! Set `LLM_PROVIDER=ollama` and `OLLAMA_URL=http://localhost:11434` in `.env`
